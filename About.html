MusicSound

About the website:

MusicSound is responsive for all mobile phones and tablets. I have used the principles of HCI. I used large sized letters because on mobile platforms it gives the opportunity to users to walk and using this application at the same time. Also I used these specific colors and font because are easier to read. Moreover, i used the alternative attribute to give a name to each image in order to be accessible even for blind people. In addition i chose png instead of bitmaps images in order to decrease the network traffic. About the affordance, there is at the top and at the bottom a menu that navigates the users. Furthermore, when the user clicks to a link or on the menu, it changes color so it gives feedback to the users for their action. Lastly, there is consistency because the menu which contains all the pages of the website it is stable on top of the screen even if the user scrolls down.
